# samaramAI

A comprehensive AI-powered healthcare assistant application.

## Project Structure

This is a split repository containing both the frontend and backend applications:

- `/client` - React + Vite frontend application
- `/server` - Express + Node.js backend API
- `/deployment` - Deployment configurations for various platforms (Firebase, Render reference)
- `/docs` - Project documentation
- `/supabase` - Supabase database schema

## Getting Started

Please see the [Deployment Guide](docs/deployment.md) for instructions on running the application locally and deploying to production.
